What is this?

NCX-News+ is an application that gives you the latest news about NCX software, which is all written by IanSkinner1982.

How do I install it?

  1. Recommended method: Go to the NCX-News+ page in NCX-Core, and click the download button. It will be automatically installed by NCX-Core.
  2. You can also download the .zip archive (either from the top of the page or from GitHub) and extract it.

Requirements

You will need a 64-bit version of Windows 10 with .NET 5.0 Runtime installed. (you can install that here)
