What is it?

This is a collection of C# programs by NinjaCheetah. These programs were made to help him learn C# and figure out how to use Visual Studio.
Included apps are:

  • Color Buttons
  • C#Calc
  • Tic Tac No
  • C#Notes
  • C#Search

Requirements

You will need a 64-bit version of Windows 10 with .NET 5.0 Runtime installed. (you can install that here)
